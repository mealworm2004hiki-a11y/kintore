# 筋トレ記録・補助アプリ

自宅ダンベル環境での「美筋（V字・黄金比）」を目指す、記録・補助PWA。個人利用・ローカル完結（IndexedDB）・認証なし。

## 機能
- 記録入力（分割テンプレ展開／白紙・前回実績の引き継ぎ）
- 種目マスタ＋筋肉タグ（主1.0/副0.5のフラクショナル集計）
- 週間ボリューム集計＋MEV/MAV/MRV可視化
- プラトー検知＋突破7手法の提示
- 分割テンプレート（編集可能）
- 進捗グラフ（種目別重量／筋肉別ボリューム）
- メソサイクル（蓄積期／デロード自動表示）
- コンディショニングログ（薪割り等・集計外）
- kinnik CSVインポート／JSONバックアップ

## 使い方
`index.html` を開くだけ。ビルド不要の単一HTML＋Service Worker のPWA。
GitHub Pages 等の静的ホスティングにフォルダごと置けば動作します。

## 技術
- 単一 `index.html`（フレームワークなし）
- IndexedDB（端末内保存）
- Service Worker（オフライン対応）

デザイン：グラファイト背景 `#14171a` ／ 錆オレンジ `#c1632b` ／ パティーナ緑 `#4f8072`、Oswald＋Noto Sans JP＋JetBrains Mono。
